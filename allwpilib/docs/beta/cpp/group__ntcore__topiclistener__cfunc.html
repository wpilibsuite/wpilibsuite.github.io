◆ NT_CreateTopicListenerPoller()

NT_TopicListenerPoller NT_CreateTopicListenerPoller ( NT_Inst  inst)

Creates a topic listener poller.

A poller provides a single queue of poll events. Events linked to this poller (using NT_AddPolledTopicListener()) will be stored in the queue and must be collected by calling NT_ReadTopicListenerQueue(). The returned handle must be destroyed with NT_DestroyTopicListenerPoller().

Parameters
instinstance handle
Returns
poller handle
