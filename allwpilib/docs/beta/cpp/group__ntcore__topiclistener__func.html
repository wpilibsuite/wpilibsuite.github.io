◆ CreateTopicListenerPoller()

NT_TopicListenerPoller nt::CreateTopicListenerPoller ( NT_Inst  inst)

Creates a topic listener poller.

A poller provides a single queue of poll events. Events linked to this poller (using AddPolledTopicListener()) will be stored in the queue and must be collected by calling ReadTopicListenerQueue(). The returned handle must be destroyed with DestroyTopicListenerPoller().

instinstance handle
Returns
