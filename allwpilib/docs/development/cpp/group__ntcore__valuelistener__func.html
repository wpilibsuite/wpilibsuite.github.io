◆ CreateValueListenerPoller()

NT_ValueListenerPoller nt::CreateValueListenerPoller ( NT_Inst  inst)

Create a value listener poller.

A poller provides a single queue of poll events. Events linked to this poller (using AddPolledValueListener()) will be stored in the queue and must be collected by calling ReadValueListenerQueue(). The returned handle must be destroyed with DestroyValueListenerPoller().

Parameters
instinstance handle
Returns
poller handle
