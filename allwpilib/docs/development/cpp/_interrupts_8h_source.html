51/**
52 * Returns the timestamp for the rising interrupt that occurred most recently.
53 *
54 * This is in the same time domain as HAL_GetFPGATime(). It only contains the
55 * bottom 32 bits of the timestamp. If your robot has been running for over 1
56 * hour, you will need to fill in the upper 32 bits yourself.
57 *
58 * @param[in] interruptHandle the interrupt handle
59 * @param[out] status Error status variable. 0 on success.
