39 /**
40 * Start automatically capturing images to send to the dashboard.
41 *
42 * You should call this method to see a camera feed on the dashboard. If you
43 * also want to perform vision processing on the roboRIO, use getVideo() to
44 * get access to the camera images.
45 *
46 * The first time this overload is called, it calls StartAutomaticCapture()
47 * with device 0, creating a camera named "USB Camera 0". Subsequent calls
48 * increment the device number (e.g. 1, 2, etc).
49 */
50 static cs::UsbCamera StartAutomaticCapture();

52 /**
53 * Start automatically capturing images to send to the dashboard.
54 *
55 * This overload calls StartAutomaticCapture() with a name of "USB Camera
56 * {dev}".
57 *
58 * @param dev The device number of the camera interface
59 */
60 static cs::UsbCamera StartAutomaticCapture(int dev);

62 /**
63 * Start automatically capturing images to send to the dashboard.

186 /**
