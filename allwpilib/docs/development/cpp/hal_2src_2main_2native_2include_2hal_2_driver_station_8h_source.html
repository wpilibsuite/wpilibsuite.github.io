173/**
174 * Returns the approximate match time.
175 *
176 * The FMS does not send an official match time to the robots, but does send
177 * an approximate match time. The value will count down the time remaining in
178 * the current period (auto or teleop).
179 *
180 * Warning: This is not an official time (so it cannot be used to dispute ref
181 * calls or guarantee that a function will trigger before the match ends).
182 *
183 * The Practice Match function of the DS approximates the behavior seen on
184 * the field.
185 *
186 * @param[out] status the error code, or 0 for success
187 * @return time remaining in current match period (auto or teleop)
188 */
