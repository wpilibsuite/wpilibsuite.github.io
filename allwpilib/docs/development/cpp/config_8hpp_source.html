137/// The mode of the automatic \ref wpi::memory::temporary_stack creation.
138/// Set to `2` to enable automatic lifetime management of the per-thread stack through nifty counter.
139/// Then all memory will be freed upon program termination automatically.
140/// Set to `1` to disable automatic lifetime managment of the per-thread stack,
141/// requires managing it through the \ref wpi::memory::temporary_stack_initializer.
142/// Set to `0` to disable the per-thread stack completely.
143/// \ref get_temporary_stack() will abort the program upon call.
