16/**

53 /**
54 * Wait for the connection listener queue to be empty. This is primarily
55 * useful for deterministic testing. This blocks until either the connection
56 * listener queue is empty (e.g. there are no more events that need to be
57 * passed along to callbacks or poll queues) or the timeout expires.

61 * @return False if timed out, otherwise true.
