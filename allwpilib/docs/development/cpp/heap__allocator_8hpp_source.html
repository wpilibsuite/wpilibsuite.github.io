24 /// Allocates heap memory.
25 /// This function is used by the \ref heap_allocator to allocate the heap memory.
26 /// It is not defined on a freestanding implementation, a definition must be provided by the library user.
27 /// \requiredbe This function shall return a block of uninitialized memory that is aligned for \c max_align_t and has the given size.
28 /// The size parameter will not be zero.
29 /// It shall return a \c nullptr if no memory is available.
30 /// It must be thread safe.
31 /// \defaultbe On a hosted implementation this function uses OS specific facilities, \c std::malloc is used as fallback.
32 /// \ingroup allocator

35 /// Deallocates heap memory.
36 /// This function is used by the \ref heap_allocator to allocate the heap memory.
37 /// It is not defined on a freestanding implementation, a definition must be provided by the library user.
38 /// \requiredbe This function gets a pointer from a previous call to \ref heap_alloc with the same size.
39 /// It shall free the memory.

42 /// \defaultbe On a hosted implementation this function uses OS specific facilities, \c std::free is used as fallback.
