◆ NT_CreateValueListenerPoller()

NT_ValueListenerPoller NT_CreateValueListenerPoller ( NT_Inst  inst)

Create a value listener poller.

A poller provides a single queue of poll events. Events linked to this poller (using NT_AddPolledValueListener()) will be stored in the queue and must be collected by calling NT_ReadValueListenerQueue(). The returned handle must be destroyed with NT_DestroyValueListenerPoller().

Parameters
instinstance handle
Returns
poller handle
