unsigned int nt::Event::flags {0}

Event flags (NT_EventFlags).

Also indicates the data included with the event:

  • NT_NOTIFY_CONNECTED or NT_NOTIFY_DISCONNECTED: GetConnectionInfo()
  • NT_NOTIFY_PUBLISH, NT_NOTIFY_UNPUBLISH, or NT_NOTIFY_PROPERTIES: GetTopicInfo()
  • NT_NOTIFY_VALUE, NT_NOTIFY_VALUE_LOCAL: GetValueData()
  • NT_NOTIFY_LOGMESSAGE: GetLogMessage()
