Detailed Description

\effects Allocates memory using the underlying If n is 1, it will call allocate_node(sizeof(T), alignof(T)), otherwise allocate_array(n, sizeof(T), alignof(T)).

\effects Deallcoates memory using the underlying It will forward to the deallocation function in the same way as in allocate().

Returns
A pointer to a memory block suitable for n objects of type T.
Exceptions
Anythingthrown by the RawAllocator.

\requires The pointer must come from a previous call to allocate() with the same n on this object or any copy of it.
