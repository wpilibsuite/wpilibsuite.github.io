140/**

142 *
143 * Sets the maximum period where the device is considered moving. This value is
144 * used to determine the "stopped" state of the encoder using the

146 *
147 * @param[in] encoderHandle the encoder handle

155/**
156 * Determines if the clock is stopped.
157 *
158 * Determines if the clocked input is stopped based on the MaxPeriod value set
159 * using the SetMaxPeriod method. If the clock exceeds the MaxPeriod, then the
160 * device (and encoder) are assumed to be stopped and it returns true.
161 *
162 * @param[in] encoderHandle the encoder handle
163 * @param[out] status Error status variable. 0 on success.
164 * @return true if the most recent encoder period exceeds the MaxPeriod value
165 * set by SetMaxPeriod
166 */

168 int32_t* status);
