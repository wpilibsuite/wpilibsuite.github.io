inlinenoexcept
Effects:
Creates it from a reference to a RawAllocator. It will store an allocator_reference to it. \requires The expression allocator_reference<RawAllocator>(alloc) is well-formed, that is either RawAlloc is the same as RawAllocator or RawAllocator is the tag type any_allocator. If the requirement is not fulfilled this function does not participate in overload resolution.
Note
The caller has to ensure that the lifetime of the RawAllocator is at least as long as the lifetime of this std_allocator object.
