template<std::size_t Size>
wpi::memory::static_allocator::static_allocator ( static_allocator_storage< Size > &  storage)
inlinenoexcept

\effects Creates it by passing it a static_allocator_storage by reference.

It will take the address of the storage and use its memory for the allocation. \requires The storage object must live as long as the allocator object. It must not be shared between multiple allocators, i.e. the object must not have been passed to a constructor before.
