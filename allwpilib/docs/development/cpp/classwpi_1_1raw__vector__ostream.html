Detailed Description

A raw_ostream that writes to a vector.

This is a simple adaptor class. This class does not encounter output errors. raw_vector_ostream operates without a buffer, delegating all memory management to the vector. Thus the vector is always up-to-date, may be used directly and there is no need to call flush().
